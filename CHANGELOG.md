## [Unreleased]

### Added
- Initial release of Jetta JWT CLI tool
- JWT decoding with support for all major algorithms (HMAC, RSA, ECDSA, EdDSA)
- JWT validation with configurable options
- Multiple input methods: CLI argument, file, or stdin
- Human-readable colorful output format
- JSON output format (--json flag)
- Humanized time display for token expiration (e.g., "8 years ago" instead of raw seconds)
- Comprehensive error messages
